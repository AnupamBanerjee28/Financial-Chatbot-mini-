## **Financial-Chatbot-mini**

- Built a financial insights chatbot using Python to provide instant responses on revenue, assets, liabilities, and net income data.

- Designed and implemented a modular chatbot architecture with clean function separation for maintainability and scalability.

## **Features**

- Answers financial queries for companies like Tesla, Microsoft, and Apple

- Provides metrics including revenue, net income, assets, and liabilities

- Command-line interface with continuous query loop

- Lightweight and easy to extend with additional financial data

## **Technologies & Tools Used**

Programming Language	Python 3

Core Libraries	Built-ins only (no external dependencies)

Development Tools	Jupyter Notebook, VS Code
